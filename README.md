# SC2002 2025S1 Assignment
## Internship Placement Management System
The Internship Placement Management System(IPMS) is a Java application designed to provide a common medium for companies, students, employers, and career center staff to efficiently manage the process of internship placements.
### Run instructions:
Run the following command in terminal in src folder to start the application.
```
java -cp bin app.Main
```
